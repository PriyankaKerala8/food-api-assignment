# 🍽️ Food API Assignment – Node.js

This project fetches data from a public **food API** (TheMealDB) using three different JavaScript methods:

- 📞 Callback
- 🔮 Promise
- ⚡ Async/Await

## 🔧 Technologies Used

- Node.js
- Axios
- Git + GitHub

1. Clone the repo or download ZIP
